Getting Started: Your First Commands

Open your terminal and try these fundamental commands. Each is safe to run:

Linux Terminal
# Find out where you are pwd # See what's in your current directory ls ls -la # Detailed view with hidden files # Navigate directories cd /home cd ~ # Go to your home directory cd .. # Go up one directory cd - # Go back to previous directory

Practice file management with these commands:

File Operations

File Operations Mastery

Learn to copy, move, and organize files:

Advanced File Operations
# Copy files and directories cp greeting.txt backup_greeting.txt cp -r my_test_folder my_backup_folder # Move and rename files mv greeting.txt welcome.txt mv welcome.txt my_test_folder/ # Remove files and directories (be careful!) rm backup_greeting.txt rm -r my_backup_folder # Remove directory and contents # Find files find . -name "*.txt" find /home -name "welcome.txt"

Text Processing Power

Search and manipulate text like a pro:

Text Processing
# Search within files grep "Hello" *.txt grep -r "Linux" /home/username/ # Count lines, words, characters wc filename.txt wc -l filename.txt # Just count lines # Sort and filter sort filename.txt sort -r filename.txt # Reverse order uniq filename.txt # Remove duplicates # View file contents head filename.txt # First 10 lines tail filename.txt # Last 10 lines tail -f logfile.txt # Follow file changes

System Monitoring

Keep track of your system's health:

System Information
# System information uname -a # System information whoami # Current username date # Current date and time uptime # System uptime and load # Process management ps aux # Show all running processes top # Real-time process viewer (q to quit) htop # Enhanced process viewer (if installed) # Disk and memory usage df -h # Disk space usage du -h # Directory space usage free -h # Memory usage

Package Management Commands

Install and manage software efficiently:

📦 Ubuntu/Debian (APT)

# Update package database sudo apt update # Install software sudo apt install git python3 curl # Remove software sudo apt remove package_name sudo apt autoremove # Search packages apt search python

🎩 Red Hat/Fedora (DNF)

# Update system sudo dnf update # Install software sudo dnf install git python3 curl # Remove software sudo dnf remove package_name # Search packages dnf search python

📱 Universal (Snap)

# Install snap packages sudo snap install code sudo snap install discord # List installed snaps snap list # Update snaps sudo snap refresh

🌐 Network Commands

# Test connectivity ping google.com ping -c 4 google.com # Download files wget https://example.com/file.txt curl -O https://example.com/file.txt # Network info ifconfig ip addr show

🎯 Practice Challenge: Create a Backup Script

Your mission:

  1. Create a new file called backup.sh
  2. Make it executable
  3. Have it copy all .txt files to a daily_backup folder
  4. Add a timestamp to the backup folder name
💡 Click here for the solution (try it yourself first!)
#!/bin/bash backup_dir="daily_backup_$(date +%Y%m%d)" mkdir -p "$backup_dir" cp *.txt "$backup_dir"/ 2>/dev/null || echo "No .txt files found" echo "Backup completed to $backup_dir" # To use: # 1. Save this as backup.sh # 2. chmod +x backup.sh # 3. ./backup.sh

Advanced Linux Concepts

🔐 Permissions & Security

# Change file permissions chmod 755 script.sh # rwxr-xr-x chmod +x script.sh # Add execute # Change ownership sudo chown user:group file.txt # View permissions ls -la

⚙️ Process Management

# View processes ps aux | grep python # Kill processes kill PID killall process_name # Run in background command & nohup command &

🔧 System Services

# Manage services (systemd) sudo systemctl start service sudo systemctl stop service sudo systemctl status service sudo systemctl enable service

📊 Monitoring & Logs

# View system logs journalctl -f tail -f /var/log/syslog # System monitoring htop iotop # I/O monitoring nethogs # Network monitoring
